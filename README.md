# Project: Antigravity Lean Team

## 概要
AIエージェントによる、自己進化型・最小構成開発プロジェクト。
「コードの肥大化」というAI開発最大の弱点を克服し、筋肉質なプロダクトを高速生成する。

## チーム構成
- **Strategist**: ロードマップ策定とスキル拡張。
- **Tech Lead**: 資源最適化とアーキテクチャ管理。
- **QA Audit**: 重複排除と品質監査。
- **Coder/Debugger/Reviewer**: 実装と堅牢化の実行。

## 運用ルール
全ての成果物は `.antigravity/skills/` に定義された「筋肉質開発スキル」に基づき検品される。
