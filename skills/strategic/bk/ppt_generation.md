# Skill: Logical PPT Presentation Generation

## 役割
ロードマップや戦略の根拠を、視覚的に理解しやすいPowerPoint形式（またはスライド構成）で出力する。

## 実行プロセス
1. **スライド構成（Ghost Deck）の作成**: 各スライドのタイトル、メッセージ、キービジュアル案をテキストで定義する。
2. **テンプレートの選択・適用**: プロジェクト指定のテンプレート（マスター）に基づき、フォント、配色、レイアウトを統一する。
3. **図解・グラフの挿入**: データの推移やロジックの構造を、図解（SmartArt的構造）やグラフとして定義する。
4. **ファイル生成**: python-pptx等のライブラリ、またはAntigravityのArtifacts機能を用いて、編集可能な.pptxファイルを出力する。
