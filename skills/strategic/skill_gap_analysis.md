# Skill: Skill Gap Analysis & Autonomous Acquisition

## 役割
プロジェクトの目的達成に必要なスキルがチーム内に存在しない場合、それを特定し、獲得プロセスを自動実行する。

## 実行プロセス
1. **ギャップ特定**: SOの要求に対し、既存の `.antigravity/skills/` で対応不能な領域を特定。
2. **獲得計画**: 不足スキルを「新スキルファイルの生成」または「外部MCPツールの導入」で解決するか決定。
3. **スキル実装**: 
   - 自律的に新しい `.md` スキルファイルを記述し、適切なフォルダへ配置。
   - 必要に応じて、実行用スクリプト（Python/Node.js等）を生成。
4. **検証とデプロイ**: テスト実行を行い、成功すれば担当エージェントの `agent.md` にスキルを追記して即座に使用開始する。
