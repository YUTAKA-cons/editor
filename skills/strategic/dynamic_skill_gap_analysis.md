# Skill: Dynamic Skill Gap Analysis
## 概要
未知の要求に対し、チーム内の既存スキル（.md）で対応不能な領域を特定し、自律的に新しい「武器」を獲得・実装する。

## 実行プロセス
1. **ギャップ特定（Audit）**:
   - SOの要求と既存の `skills/` を照合し、不足している知識や規約をリストアップする。
2. **スキルファイルの自動生成**:
   - 不足分を補うための汎用的な `.md` ファイルを自ら記述し、適切なディレクトリに保存する。
3. **エージェントへのオンボーディング**:
   - 新スキルを各エージェントの `agent.md` に紐付け、即座に実行可能な状態へアップデートする。

## 成功基準
- チームが「できない」と言わず、数分後には「新しいスキルを獲得したので実行可能です」と回答できること。
